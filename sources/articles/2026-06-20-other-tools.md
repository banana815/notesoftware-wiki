# 其他值得关注的笔记工具

## Bear
- **定位**: Apple-only 极简书写体验
- **组织**: 标签系统（嵌套 #tags），非文件夹
- **平台**: macOS/iOS/iPadOS/Watch/visionOS
- **价格**: Pro $2.99/mo ($29.99/yr)
- **导出**: 极佳 — PDF, DOCX, HTML, JPG, ePub, Markdown, TextBundle
- **缺少**: AI、协作、Web Clipper、非 Apple 平台
- **胜在**: 专注和精致

## Craft
- **定位**: Block 编辑器，Apple 为主 + Windows + Web
- **AI**: Llama 3.2 / GPT-4o / DeepSeek R1 / 自定义 API Key
- **功能**: 实时协作、白板、任务、Collections（数据库）
- **价格**: 免费 (1,500 blocks), Plus ~$4.79/mo
- **关键缺口**: 无 Android

## Amplenote
- **定位**: 笔记 + 任务 + 日历统一
- **核心**: Task Score（Eisenhower 优先级），Google Calendar/Outlook 同步
- **AI**: Ample Agent Pro (LLM 集成, April 2026)
- **价格**: Free, Pro $5.84/mo, Unlimited $10/mo
- **限制**: 反向链接不如 Obsidian 精致，主要是单人生产力工具

## RemNote
- **定位**: 笔记 + 间隔重复
- **核心差异**: 唯一原生 SM-2 + FSRS-4.5 的笔记工具
- **内联闪卡**: 在任意 bullet 后输入 `::` 即创建闪卡
- **AI**: PDF AI 闪卡生成
- **价格**: Free (有限 AI), Pro $8/mo, Pro+AI $18/mo
- **限制**: 移动端弱、学习曲线陡

## UpNote
- **定位**: 最佳性价比的跨平台笔记
- **价格**: **$39.99 终身许可** — 反订阅制的独特选择
- **平台**: Mac, Windows, Linux, iOS, Android
- **定位**: 正在从 Evernote 流失用户中获取份额

## Reflect
- **定位**: 极简 AI 原生、Apple 生态
- **AI**: GPT-4o / Claude Sonnet，自定义 AI Prompts
- **安全**: E2EE 但云端
- **价格**: $10/mo，无免费层
- **背景**: 前 Clearbit 创始人 Alex MacCaw，避免 VC，Wefunder 众筹

## Mem.ai
- **定位**: AI Thought Partner
- **融资**: ~$29M (a16z, OpenAI Startup Fund), $110M 估值
- **2023 ARR**: $4.2M
- **Mem 2.0 (Oct 2025)**: 重构为"世界首个 AI 思维伙伴"，现支持本地优先 + 离线
- **核心**: 上下文记忆侧栏自动浮现相关信息

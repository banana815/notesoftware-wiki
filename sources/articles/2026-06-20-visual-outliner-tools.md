# 视觉化与大纲式笔记工具

## Workflowy
- 原始大纲工具
- 无限嵌套 bullet 列表
- Freemium: 250 条目免费，$5-10/mo 无限
- 正被功能更丰富的新工具蚕食市场份额

## Dynalist
- Workflowy 原始创始人打造的增强版本
- 完整 Markdown 支持
- Freemium
- 被称为"Workflowy 超级增强版"

## Milanote
- 视觉化 Mood Board 风格笔记
- 面向设计师、营销人员、创意团队
- 有限免费 boards，Pro $10/mo
- 非 PKM 工具，而是创意协作空间

## Scrintal
- 视觉化笔记 + 嵌套卡片 + 白板
- 定位为"视觉思考者的 Obsidian 替代品"
- Pro ~$10-15/mo
- Freemium

## Supernotes
- 社交/协作笔记
- 卡片 + 嵌套层级
- Freemium (~$6/mo Pro)
- 利基多人角度 — 为共享知识库设计

## Heptabase（再次强调）
- 卡片 + 白板为核心隐喻
- Pro $8.99/mo, Premium $17.99/mo
- AI Tutor, Zotero 集成
- 最完整的视觉 PKM 方案

## 细分市场定位

| 工具 | 目标用户 | 核心隐喻 |
|------|---------|---------|
| Workflowy/Dynalist | 大纲爱好者 | 无限嵌套列表 |
| Milanote | 创意专业人士 | Mood Board |
| Scrintal | 视觉思考者 | 卡片+白板 |
| Supernotes | 协作团队 | 社交卡片 |
| Heptabase | 学术研究者 | 白板+元认知 |

# Obsidian

- **定位**: 本地优先、可扩展的 Markdown PKM
- **用户**: 120万+
- **商业模式**: 核心免费，Sync $4/mo，Publish $8-16/mo。未融资，独立盈利。
- **平台**: macOS, Windows, Linux, iOS, Android

## 核心特性
- 纯本地 Markdown 文件 — 你拥有数据
- 双向链接 + 图谱视图
- Canvas 白板（1.12+ 支持反向链接）
- Bases (v1.7+) — YAML frontmatter 的 GUI 数据库层
- 2,500+ 社区插件 — 最丰富的 PKM 插件生态
- 2026 新增 CLI、Canvas 反向链接、移动端功能对等

## 优势
- 数据可移植性最佳（纯 .md 文本）
- 插件生态最成熟（TypeScript API）
- 社区大且活跃
- 完全的离线能力

## 劣势
- 无原生 AI（依赖社区插件）
- 无原生协作
- 同步需要付费或 DIY 配置
- 空白画布式入门令新手不知所措
- 移动端历史体验不佳（2026 在改进）
- 插件废弃可能破坏工作流

## 竞争定位
本地优先 PKM 的标杆。适合重视数据所有权、喜欢自定义的技术用户。不适合需要实时协作或零配置 AI 的用户。

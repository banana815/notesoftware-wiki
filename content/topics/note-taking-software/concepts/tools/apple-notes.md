# Apple Notes

- **定位**: Apple 生态内置笔记，2025-2026 正从"备忘录"进化为全功能笔记工具
- **平台**: iOS, iPadOS, macOS, watchOS, Web (iCloud.com)
- **价格**: 免费（随 iCloud 存储）

## 编辑体验
- 原生 WYSIWYG，极速流畅
- Apple Pencil 手写支持（Reed Pen 书法工具）
- 内联绘图、文档扫描、表格
- Quick Notes（任意 app 滑动呼出）
- iOS 26 自适应工具栏（18 个按钮）

## 文件管理
- iCloud 自动同步
- 文件夹 + 智能文件夹 + 标签
- Spotlight 集成搜索
- 文档扫描 + OCR
- iOS 26+ 支持 Markdown 导入/导出
- iOS 26 音频录制 + 实时转录 + AI 摘要
- iOS 27 Markdown 粘贴自动转为富文本 / "Copy as Markdown"

## 优势
- 完全免费
- 最佳 Apple 生态集成（Quick Notes, Share Sheet, Siri）
- 零学习成本
- 近年更新速度显著加快（iOS 26/27 是分水岭）

## 劣势
- Apple 生态锁定（无 Windows/Android/Linux）
- 无双向链接、无图谱
- 无社区插件
- 无数据库/表格视图
- 大规模知识库管理能力有限

## 定位
最好的"入门级"笔记工具。如果你活在 Apple 生态且不需要高级 PKM 功能，Apple Notes 已经足够。iOS 26/27 的更新正缩小与专业工具的功能差距。

# TriliumNext

- **定位**: 最强大的自托管层级笔记，笔记克隆是杀手级特性
- **许可**: 开源（原项目停止维护，TriliumNext 社区活跃）
- **部署**: 单 Docker 容器，~100-300MB RAM
- **平台**: 桌面 (Electron) + Web/PWA（无原生移动端）

## 编辑体验
- 富文本 WYSIWYG + Markdown/代码笔记
- 代码片段（语法高亮）、表格、图片
- 层级树组织，类似文件管理器但更智能

## 文件管理
- 单 SQLite 加密数据库 — 备份即复制文件
- **笔记克隆**: 同一笔记可存在于多个位置（类似符号链接）
- 自托管同步（桌面 ↔ 服务器双向同步）
- 完整修订历史（一键快照，任意回滚）
- 内置 JS 脚本引擎 (ETAPI)
- 关系图（类似 Obsidian Graph View）
- 公开笔记分享
- Markdown/HTML/单文件备份导出

## 优势
- 笔记克隆 — 编辑一次，所有位置更新（最独特的特性）
- 极低资源消耗（Raspberry Pi 可运行）
- 内置脚本引擎无需插件生态
- 自托管 = 完全数据控制

## 劣势
- **无原生移动端**（仅 PWA，编辑体验差）
- 单用户（无团队协作）
- SQLite 存储不是纯 Markdown 文件
- 社区较小
- UI 偏功能实用型

## 竞争定位
最适合重视数据控制、层级思维、愿意自托管的技术用户。笔记克隆功能独一无二—完美适用于信息跨多个分类的场景。

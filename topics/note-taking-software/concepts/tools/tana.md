# Tana

- **定位**: AI 原生的 Supertag 节点图数据库
- **用户**: 16万+ waitlist (2025)，Fortune 500 采用
- **融资**: $25M (Tola Capital, Lightspeed, Northzone)
- **定价**: 免费层，Pro/Plus ~$14/mo，Team ~$14/seat/mo+

## 核心特性
- **Supertags** — 标签 = OOP 类，自带 Schema。`#book` 自动添加 author/status/rating 字段，支持继承
- **Live Search Nodes** — 查询作为一等节点，实时更新
- **AI Agent** — 从 Zoom/语音转录 → 自动结构化节点 → 跨图关联
- 节点图数据库 — 所有内容是可引用的节点

## 优势
- 最强大的结构化能力（Supertags）
- AI 原生架构，从转录到结构化全自动
- 社区活跃（Slack 24,000+）
- 创新性最强的新一代工具

## 劣势
- 学习曲线最陡（数周才能熟练）
- 纯云端，无离线
- JSON 导出丢 Supertag 语义 — 锁定风险高
- 小团队，长期可持续性待验证
- VC 资金 → 增长压力

## 竞争定位
Notion + Roam 的下一代替代品。适合愿意投资学习成本、相信 AI 结构化未来的 Power User。不适合需要离线、简单、或数据可移植性的用户。

# 国产笔记软件

## 语雀 (Yuque) — 蚂蚁集团

- **定位**: 企业知识库/专业文档平台
- **编辑**: WYSIWYG + Markdown，代码块/公式出色，需"发布"步骤
- **文件管理**: 目录树，版本历史完善
- **协作**: 最强 — 多人实时编辑、评论讨论
- **AI**: 阿里系 AI，文档智能处理
- **缺点**: 导出 Markdown 不完整（图片在线地址），不适合碎片记录

## Wolai (我来) — 独立团队

- **定位**: Notion 国产替代
- **编辑**: Block 编辑器，中文排版/快捷键体验最佳，**页面关系图**独家
- **文件管理**: 支持导出完整 Markdown+本地图片，离线编辑，文件预览需付费
- **缺点**: 2025 涨价争议，大笔记性能问题，免费版限制多，AI 功能弱

## FlowUs (息流) — 独立团队

- **定位**: 信息中枢/Notion 替代
- **编辑**: Block 编辑器，原生移动端极速流畅
- **文件管理**: **文件夹页面 = 内置网盘**（杀手级），多维表格 7 种视图
- **AI**: 最成熟 — PDF 对话、头脑风暴、大纲生成
- **缺点**: 早期有 bug（已改善），部分导出功能开发中

## 趋势
- Notion 中文版可能带来冲击
- Wolai 用户因涨价流失中
- FlowUs 在 AI 和移动端走在前列
- 三者都在快速迭代，数据可迁移性（Markdown 导出）成为用户选型关键考量

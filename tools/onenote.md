# Microsoft OneNote

- **定位**: 自由画布企业笔记，Microsoft 365 生态集成
- **平台**: Windows, macOS, iOS, Android, Web
- **价格**: 免费（Windows 内置），Microsoft 365 订阅可获得更多存储和 Copilot AI

## 编辑体验
- 自由画布 — 点击任意位置开始输入
- 墨迹/手写体验无可匹敌（触控笔生态最强）
- 数学助手、录音同步笔记
- AI Copilot：摘要/任务列表/改写 (Dec 2025)，理解图像/表格/手写 (April 2026)

## 文件管理
- 笔记本 → 分区 → 页面 三级结构
- OneDrive 同步
- 内建版本历史
- PDF 打印输出 + 嵌入文件
- 图像 OCR + 手写文本搜索（行业最佳）
- Purview 敏感度标签 + DLP (Jan 2026) — 企业合规

## 优势
- 完全免费（Windows）
- 最佳手写/墨迹体验
- 企业级安全和合规
- Microsoft 365 深度集成
- Copilot AI 理解多模态内容

## 劣势
- 无双向链接、无知识图谱
- 无 Markdown 原生支持
- 数据库/表格不如 Notion
- Copilot 需付费 M365 Copilot 许可 (April 2026 取消免费)
- 导出质量一般
- 同步偶尔不稳定

## 定位
企业环境的首选。手写和墨迹方面无可匹敌。适合已经使用 Microsoft 365 的组织，不适合个人 PKM 爱好者。

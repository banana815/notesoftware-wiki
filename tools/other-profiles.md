# 其他工具档案

## Bear
Apple-only 极简书写。标签组织 (嵌套 #tags)。Pro $2.99/mo。导出极佳但缺 AI/协作/Web Clipper。胜在专注和精致。

## Craft
Block 编辑器。Apple + Windows + Web（无 Android）。多模型 AI (Llama/GPT/DeepSeek)。实时协作 + 白板 + Collections。Plus ~$4.79/mo。

## Amplenote
笔记 + 任务 + 日历统一体。Task Score 优先级排序。Ample Agent Pro (April 2026)。Pro $5.84/mo。单人生产力工具。

## RemNote
唯一原生 SM-2 + FSRS-4.5 SRS 的笔记工具。`::` 内联闪卡。AI 闪卡生成。Pro+AI $18/mo。弱移动端，陡学习曲线。

## Reflect
极简 AI 原生。GPT-4o + Claude Sonnet。E2EE 但云端。$10/mo 无免费层。前 Clearbit 创始人，Wefunder 众筹。

## Mem.ai
"AI 思维伙伴"。$29M 融资 (a16z, OpenAI)。Mem 2.0 (Oct 2025) 支持本地优先+离线。自动浮现关联记忆。

## UpNote
$39.99 终身许可 — 反订阅制标杆。全平台 (Mac/Win/Linux/iOS/Android)。从 Evernote 流失用户获取份额。

## Standard Notes
最强加密（XChaCha20-Poly1305 + Argon2 层级密钥）。零知识架构。$90/yr。被 Proton 收购。

## Joplin
AGPL 开源。Markdown + SQLite。7+ 可插拔同步后端。RSA E2EE。强在多平台和同步灵活性。

## AppFlowy
Rust + Flutter 开源 Notion 替代。AGPL。Yrs CRDT。60K+ GitHub Stars。仍在早期。

## Roam Research
双向链接范式开创者。Datomic Datalog 数据库。$15/mo。产品停滞 4 年+。数据锁定风险极高。已被竞争者超越。

## Workflowy / Dynalist
原始大纲工具。无限嵌套 bullet。Freemium。Dynalist 是 Workflowy 创始人的增强版。

## Milanote
视觉 Mood Board 笔记。面向创意专业人士。Pro $10/mo。非 PKM，而是创意空间。

## Scrintal
视觉卡片 + 白板。"视觉思考者的 Obsidian"。Pro ~$10-15/mo。

## Supernotes
社交卡片笔记。协作知识库。~$6/mo Pro。利基多人场景。

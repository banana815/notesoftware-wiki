# Notion

- **定位**: 云端 all-in-one 协作平台，正转型为开发者平台
- **用户**: 3,000万+
- **商业模式**: 个人免费，Plus $10/mo，Business $20/用户/mo。VC 支持。
- **平台**: Web, macOS, Windows, iOS, Android

## 核心特性
- Block 编辑器 + 关系型数据库
- 实时多人协作
- 丰富的模板市场
- 2026: AI Agent → Custom Agents 24/7 自主运行
- Notion 3.5: Workers, Agent SDK, CLI, Markdown API → 开发者平台

## 优势
- 最佳团队协作体验
- 数据库/表格功能最强
- AI 集成最深（GPT-5, Claude, Gemini）
- 模板生态和社区最大

## 劣势
- 离线模式弱（50 条目限制、无子页面继承）
- 数据锁定 — 专有 Block DB，导出去元数据
- AI 仅限 Business 计划 ($20/用户/mo)
- 性能问题 — 大数据库卡顿，移动端 Web 套壳
- 定价上移趋势 — 个人用户正被挤出 AI 功能

## 竞争定位
云端协作的王者，正从笔记应用转型为 AI Agent 基础设施。适合团队和企业。不适合重视数据所有权或需要可靠离线的个人用户。

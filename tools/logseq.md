# Logseq

- **定位**: 开源 outliner，日记优先的 PKM
- **用户**: 50万+
- **商业模式**: 100% 免费开源 (AGPL v3)，付费 Sync (alpha)
- **平台**: macOS, Windows, Linux, iOS, Android

## 核心特性
- 块级大纲编辑器 — 每个 bullet 可独立引用
- 每日日记为核心界面
- 内置闪卡 + 间隔重复
- 白板功能
- PDF 批注
- 双轨存储: 纯文本 Markdown/Org 模式 + SQLite DB 模式 (alpha)

## 优势
- 完全开源 (AGPL)
- 日记 + 大纲的工作流非常自然
- 内置 SRS 闪卡，学习笔记一体化
- 纯文本 Markdown 可移植性好
- DB 版查询性能提升 340%

## 劣势
- 大型库性能差（>10,000 笔记）
- DB 版仍在 alpha，同步未成熟
- 插件生态远小于 Obsidian
- 不适合长文写作（outliner 范式限制）

## 竞争定位
Obsidian 的开源替代品，outliner 范式适合日记和任务管理。适合喜欢大纲式思维、重视开源的 Power User。
